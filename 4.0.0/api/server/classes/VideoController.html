Defined in app\server\controllers\VideoController.js:399

Syntax

getEntitiesAction

(
  • request
  • response
  • next
)
async

Summary

Gets published videos by properties.

Parameters:

  • request Request

    ExpressJS HTTP Request

    • query Object

      Request's query parameters

      • query String
        To search on both videos' title and description
      • states String | Array
        To filter medias by state
      • dateStart String
        To filter medias after or equal to a date (in format mm/dd/yyyy)
      • dateEnd String
        To get medias before a date (in format mm/dd/yyyy)
      • categories String | Array
        To filter medias by category
      • groups String | Array
        To filter medias by group
      • sortBy String
        To sort medias by either **title**, **description** or **date**
      • sortOrder String
        Sort order (either **asc** or **desc**)
      • page String
        The expected page
      • limit String
        To limit the number of medias per page. If not specified get all medias
      • properties Object
        A list of properties with the property id as the key and the expected property value as the value
  • response Response

    ExpressJS HTTP Response

  • next Function

    Function to defer execution to the next registered middleware

Example:

{
  "videos" : [
    ...
  ]
}
